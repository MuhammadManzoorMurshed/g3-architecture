# G3 Architecture

A modern, fully responsive landing page template for an architecture agency. This project is built with HTML5 and vanilla CSS3, featuring a mobile-first design, interactive UI, and responsive layouts across all devices (from 320px screens up to large desktops).

## 🚀 Live Demo

[Live Demo](https://g3-architecture-bd.netlify.app/)

## ✨ Features

- **Fully Responsive Design**: Works seamlessly on mobile phones, tablets, and desktop computers.
- **Modern UI/UX**: Clean and professional look tailored for architecture firms.
- **Mobile-First Approach**: Optimized for small screens first, progressively enhanced for larger displays.
- **No Frameworks**: Built using pure HTML5 and CSS3 (no Bootstrap, Tailwind, etc.).
- **Cross-Browser Compatible**: Works perfectly on Chrome, Firefox, Safari, and Edge.
- **Custom Fonts**: Uses 'Work Sans' and other Google Fonts for modern typography.

## 🛠️ Technologies Used

- **HTML5**: For semantic structuring of the web page.
- **CSS3**: For styling, Flexbox, CSS Grid, and responsive Media Queries.

## 💻 How to Run Locally

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/MuhammadManzoorMurshed/g3-architecture.git
   ```
2. Navigate to the project directory:
   ```bash
   cd g3-architecture
   ```
3. Open `index.html` in your favorite web browser or use an extension like **Live Server** in VS Code.
